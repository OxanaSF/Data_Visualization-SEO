Data Visualization with D3.js

D3 stands for Data-Driven Documents. It is an open-source JavaScript data visialization library developed by Mike Bostock to create custom interactive data visualizations in the web browser using SVG, HTML and CSS. Here you can find the useful article about D3.js.

On the right, you can see the bar chart I created using the d3js library. It represents crime rate of five districts in San Francisco with the highest crime rating in a descending order. Put mouse over bars to see numbers.

Purpose:

This graphic is created to show the five districts in San Francisco with the hieghest crime rating in the period from 2018 to present (data is taken on September 17, 2020); and demonstrate how knowledge of d3.js couldbe applied.

Source

This image is based on data from the San Francisco Open data Police Department Incident Reports: 2018 to Present you can find here.

Data Visualization Tools

The aggregation of the data from a CSV file was done Python. The creation on the bar chart was done with d3.js, version5.15 (the latest version (6.1.1)). Full information about d3.js could be found on its oficial website. d3js.org.
